# Skills Archivadas

Las skills archivadas han sido removidas del repo. El catálogo completo con sus fuentes y motivo de archivo está en el **README.md → Sección 8.7**.

## Resumen

Todas las skills archivadas provienen de dos fuentes:
- **`garrytan/gstack`** — se reinstalan con el mismo `git clone + ./setup`
- **`Yhoswar/claude-skills`** — se reinstalan via `/install-skill yhosw-skills` o copiando desde las carpetas del repo

## Cómo restaurar una skill archivada

1. Identificar la fuente en [README sección 8.7](README.md#87-skills-archivadas--solo-si-se-necesitan)
2. Reinstalar desde la fuente original
3. Reiniciar Claude Code

> No se guardan copias locales — instalar desde fuente es más limpio y garantiza la versión actualizada.
